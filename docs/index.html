Welcome to the Sudoku Solver!

How to use:

Enter the Sudoku board and click the "Solve Sudoku" button to solve the puzzle.

You can enter the Sudoku board in multiple ways:

  • Interactively by clicking on the cells: Enter a number between 1 and 9 in each cell.
  • Pasting on the board: Pasted numbers from the clipboard and they will be distributed across the cells.
  • Use the input field: Enter the board or paste it in the input field. There are multiple formats supported:
    • Single line of numbers: Enter the board as a single line of numbers, with dots (.) representing empty cells.
    • 2D array format: Enter the board as a 2D array of numbers, with empty cells represented as null or 0.

You can also edit the board by clicking on the cells or using the input field.
